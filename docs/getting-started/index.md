# Getting Started

Welcome to RHAPSODY! This section will guide you from installation to running your first heterogeneous workflows.

## Learning Path

1.  **[Installation](installation.md)**: Set up your environment and install RHAPSODY with the necessary backend supports.
2.  **[Quick Start](quick-start.md)**: Run a simple local workflow to understand the core Session/Task/Backend objects.
3.  **[Advanced Usage](advanced-usage.md)**: Learn how to route tasks to multiple backends and mix HPC with AI inference.
4.  **[Configuration](configuration.md)**: Deep dive into tuning individual backends for your specific infrastructure.

!!! note "Prerequisites"
    RHAPSODY is built on top of `asyncio`. Familiarity with Python's `async/await` syntax is highly recommended.

## Next Steps

Once you've mastered the basics, you can explore the full **[API Reference](../api/)** for detailed module and class documentation.
